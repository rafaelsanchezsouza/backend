## Getting Started

### Requirements

- NPM / Yarn

```
npm install -g yarn
```

### Instalation

1. Clone Repository

```
git clone https://github.com/rafaelsanchezsouza/ecommerce-back-end
```

2. Install required packages

```
yarn install
```

3. Start local application

```
yarn dev
```
